# BiliVideoData

an android apk about view [BiLiBiLi](https://www.bilibili.com/)-VideoData

## Installation - 安装
 - [Download Latest](https://github.com/TANGhz17/BiliVideoData/releases)
 - Install on your phone

## Changelog - 更新日志
### 1.0
  - 实现了查看B站视频详细信息的功能
### 准备更新的功能：
 - 视频信息查看界面变为分界面，设置主界面
 - 添加评论区抽奖功能
 - 绑定使用者与up主，获取个人视频数据
 - 使用后台数据存储，生成数据曲线
